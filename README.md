## What is Kaggle Learn?

Kaggle Learn is a free, hands-on platform for learning data science, machine learning, and AI. Every course is built around interactive Python notebooks run directly in the browser — no setup required. Courses are short, practical, and project-focused.

---

## Structure

Each course lives in its own subdirectory and follows this layout:

```
CourseTitle/
├── README.md        # Course overview, notes, and progress
└── *.ipynb          # Jupyter notebooks — one per lesson
```
